01அறிமுகம் — தரவுக் கட்டமைப்பு என்றால் என்ன?

ஒரு மாறியில் (variable) ஒரே ஒரு பெறுமானத்தையே சேமிக்க முடியும். ஆனால் நடைமுறையில் — 40 மாணவர்களின் புள்ளிகள், ஒரு வாரத்தின் வெப்பநிலைகள், பொருட்களும் விலைகளும் — பல பெறுமானங்களை ஒன்றாகச் சேமித்துக் கையாள வேண்டியுள்ளது. இதற்காக Python வழங்கும் கருவிகளே தரவுக் கட்டமைப்புகள் (Data Structures).

  • சரம் (String) — எழுத்துகளின் வரிசை. எ.கா: "Jaffna"
  • பட்டியல் (List) — மாற்றக்கூடிய (mutable) உறுப்புகளின் வரிசை. எ.கா: [45, 78, 62]
  • டியூப்பிள் (Tuple) — மாற்ற முடியாத (immutable) உறுப்புகளின் வரிசை. எ.கா: (9.66, 80.01)
  • அகராதி (Dictionary) — சாவி : பெறுமானம் (key : value) சோடிகளின் தொகுப்பு. எ.கா: {"si": "Sinhala", "ta": "Tamil"}
இரு முக்கிய பண்புகள் வரிசைமுறை (ordered) — உறுப்புகள் இட எண் (index) மூலம் அணுகப்படுமா?
மாற்றத்தக்கமை (mutable) — உருவாக்கிய பின் உறுப்புகளை மாற்ற முடியுமா?
இவ்விரு பண்புகளின் அடிப்படையிலேயே நான்கு கட்டமைப்புகளும் வேறுபடுகின்றன.
02அட்டவணைப்படுத்தல் (Indexing) & துண்டாக்கல் (Slicing)

சரம், பட்டியல், டியூப்பிள் மூன்றும் வரிசைகள் (sequences); அவற்றின் உறுப்புகள் இட எண்களால் அணுகப்படுகின்றன. இட எண் 0 இலிருந்து தொடங்கும்; எதிர்ம இட எண்கள் இறுதியிலிருந்து எண்ணப்படும்:

word = "COMPUTER"
print(word[0])      # C   (முதல் எழுத்து)
print(word[3])      # P
print(word[-1])     # R   (கடைசி எழுத்து)
print(word[2:5])    # MPU (இட 2 முதல் 4 வரை — 5 சேராது!)
print(word[:4])     # COMP (தொடக்கம் முதல் இட 3 வரை)
print(word[4:])     # UTER (இட 4 முதல் இறுதி வரை)
print(len(word))    # 8   (எழுத்துகளின் எண்ணிக்கை)
பரீட்சை நுணுக்கம் துண்டாக்கலில் [start : end] — end இட எண் சேர்க்கப்படாது. word[2:5] என்பது இடங்கள் 2, 3, 4 ஐ மட்டுமே தரும். இதே விதி பட்டியலுக்கும் டியூப்பிளுக்கும் பொருந்தும்.
03சரம் (String)

எழுத்துகளின் வரிசை; ஒற்றை ' ' அல்லது இரட்டை " " மேற்கோள்களுக்குள் எழுதப்படும். சரம் மாற்ற முடியாதது (immutable) — உருவாக்கிய பின் எழுத்துகளை நேரடியாக மாற்ற முடியாது.

அடிப்படைச் செயற்பாடுகள்

a = "Hello"
b = "World"

print(a + " " + b)    # Hello World  (இணைத்தல் — concatenation)
print(a * 3)          # HelloHelloHello (மீள்பெருக்கம்)
print("ell" in a)     # True  (உள்ளடக்கச் சோதனை)
print(len(a))         # 5

முக்கிய முறைகள் (String Methods)

முறைசெயல்எடுத்துக்காட்டு (s = "Sri Lanka")
s.upper()பேரெழுத்தாக்கும்"SRI LANKA"
s.lower()சிற்றெழுத்தாக்கும்"sri lanka"
s.find("L")முதல் இடத்தைத் தரும் (இல்லாவிடின் -1)4
s.replace("Sri","SRI")பதிலீடு செய்த புதிய சரம்"SRI Lanka"
s.split(" ")பிரித்து பட்டியலாக்கும்['Sri', 'Lanka']
s.count("a")எத்தனை முறை வருகிறது2
s.strip()இருபுற வெற்றிடங்களை நீக்கும்—

சரம் முறைகள் — விரிவான எடுத்துக்காட்டுகள்

1) எழுத்து வடிவ முறைகள் (Case methods):

s = "sri lanka"

print(len(s))            # 9  ← வெற்றிடமும் ஓர் எழுத்தே!
print(s.upper())         # SRI LANKA   (எல்லாம் பேரெழுத்து)
print(s.lower())         # sri lanka   (எல்லாம் சிற்றெழுத்து)
print(s.capitalize())    # Sri lanka   (முதல் எழுத்து மட்டும் பெரிது)
print(s.title())         # Sri Lanka   (ஒவ்வொரு சொல்லின் முதலெழுத்தும்)
print("PyThOn".swapcase())  # pYtHoN  (பெரிது↔சிறிது மாற்றம்)

2) தேடல் முறைகள் (Searching):

s = "programming"

print(s.find("m"))       # 6   (முதல் m இன் இட எண்)
print(s.find("z"))       # -1  (இல்லாவிடின் -1 — பிழை இல்லை)
print(s.index("m"))      # 6   (find போன்றதே; ஆனால் இல்லாவிடின் பிழை!)
print(s.count("m"))      # 2   (m எத்தனை முறை?)
print(s.count("g"))      # 2
print(s.startswith("pro"))  # True  (இதில் தொடங்குகிறதா?)
print(s.endswith("ing"))    # True  (இதில் முடிகிறதா?)

3) மாற்றும் முறைகள் (புதிய சரத்தைத் தரும் — மூலச் சரம் மாறாது):

s = "  A/L ICT  "

print(s.strip())               # "A/L ICT"  (இருபுற வெற்றிடம் நீக்கம்)
print(s.lstrip())              # "A/L ICT  " (இடப்புறம் மட்டும்)
print(s.rstrip())              # "  A/L ICT" (வலப்புறம் மட்டும்)

t = "good morning"
print(t.replace("o", "0"))     # g00d m0rning
print(t.replace("o", "0", 1))  # g0od morning (முதல் ஒன்று மட்டும்)

csv = "1001,Amuthan,78"
parts = csv.split(",")         # சரம் → பட்டியல்
print(parts)                   # ['1001', 'Amuthan', '78']

words = ["Sri", "Lanka"]
print("-".join(words))         # Sri-Lanka  (பட்டியல் → சரம்)

4) சோதனை முறைகள் (True / False தரும்):

print("2026".isdigit())    # True  (எல்லாம் இலக்கங்களா?)
print("21A".isdigit())     # False
print("Hello".isalpha())   # True  (எல்லாம் எழுத்துகளா?)
print("AL2026".isalnum())  # True  (எழுத்து + இலக்கம் மட்டுமா?)
print("ICT".isupper())     # True  (எல்லாம் பேரெழுத்தா?)
print("ict".islower())     # True  (எல்லாம் சிற்றெழுத்தா?)
print("   ".isspace())     # True  (எல்லாம் வெற்றிடமா?)
பயன்பாடு isdigit() — பயனர் உள்ளீடு எண்ணா எனச் சரிபார்க்க (input validation) பயன்படும்: if ans.isdigit(): n = int(ans)

சரத்தினூடாக மடக்கு (Traversal)

# உயிரெழுத்துகளை (vowels) எண்ணும் நிரல்
word = input("சொல்லை உள்ளிடுக: ")
count = 0
for ch in word:
    if ch in "aeiouAEIOU":
        count = count + 1
print("உயிரெழுத்துகள் =", count)
Immutable — பரீட்சைப் பொறி s[0] = "X" → பிழை (TypeError)! சரத்தை மாற்ற, புதிய சரத்தை உருவாக்க வேண்டும்:
s = "X" + s[1:]. s.upper() போன்ற முறைகளும் s ஐ மாற்றாது — புதிய சரத்தையே திருப்பித் தரும்.
04பட்டியல் (List)

சதுர அடைப்புக்குறிகளுக்குள் [ ] எழுதப்படும், மாற்றக்கூடிய (mutable) வரிசை. வெவ்வேறு தரவு வகைகளையும் ஒரே பட்டியலில் வைக்கலாம்.

marks = [78, 85, 62, 91, 55]
mixed = [1001, "Amuthan", 78.5, True]
empty = []                  # வெற்றுப் பட்டியல்

print(marks[0])     # 78
marks[2] = 65       # மாற்றலாம்! (mutable)
print(marks)        # [78, 85, 65, 91, 55]

முக்கிய முறைகள் & சார்புகள்

முறை / சார்புசெயல்எ.கா: m = [78, 85, 62]
m.append(91)இறுதியில் சேர்க்கும்[78, 85, 62, 91]
m.insert(1, 50)இட 1 இல் நுழைக்கும்[78, 50, 85, 62]
m.remove(85)பெறுமானத்தை நீக்கும்[78, 62]
m.pop()கடைசி உறுப்பை நீக்கித் தரும்62
m.sort()ஏறுவரிசையில் ஒழுங்குபடுத்தும்[62, 78, 85]
m.reverse()தலைகீழாக்கும்[62, 85, 78]

பட்டியல் முறைகள் — விரிவான எடுத்துக்காட்டுகள்

1) சேர்க்கும் முறைகள்:

m = [10, 20]

m.append(30)           # இறுதியில் ஒரு உறுப்பு
print(m)               # [10, 20, 30]

m.insert(1, 15)        # இட 1 இல் நுழைத்தல்
print(m)               # [10, 15, 20, 30]

m.extend([40, 50])     # பல உறுப்புகளைச் சேர்த்தல்
print(m)               # [10, 15, 20, 30, 40, 50]

m.append([60, 70])     # கவனம்! பட்டியலையே ஓர் உறுப்பாகச் சேர்க்கும்
print(m)               # [10, 15, 20, 30, 40, 50, [60, 70]]
print(len(m))          # 7 (8 அல்ல!)

2) நீக்கும் முறைகள்:

m = [10, 20, 30, 20, 40]

m.remove(20)           # முதல் 20 ஐ மட்டும் நீக்கும்
print(m)               # [10, 30, 20, 40]

x = m.pop()            # கடைசி உறுப்பை நீக்கி, அதைத் தரும்
print(x, m)            # 40 [10, 30, 20]

y = m.pop(0)           # இட எண் 0 இன் உறுப்பை நீக்கித் தரும்
print(y, m)            # 10 [30, 20]

del m[0]               # இட எண் மூலம் நீக்கல்
print(m)               # [20]

m.clear()              # எல்லா உறுப்புகளையும் நீக்கல்
print(m)               # []

3) தேடல் & எண்ணும் முறைகள்:

m = [45, 78, 62, 78, 91]

print(m.index(78))     # 1   (முதல் 78 இன் இட எண்)
print(m.count(78))     # 2   (78 எத்தனை முறை?)
print(78 in m)         # True
print(100 in m)        # False
print(len(m))          # 5
print(max(m))          # 91
print(min(m))          # 45
print(sum(m))          # 354
print(sum(m)/len(m))   # 70.8  (சராசரி)

4) ஒழுங்குபடுத்தும் முறைகள்:

m = [40, 10, 30, 20]

m.sort()                    # ஏறுவரிசை — பட்டியலையே மாற்றும்
print(m)                    # [10, 20, 30, 40]

m.sort(reverse=True)        # இறங்குவரிசை
print(m)                    # [40, 30, 20, 10]

m.reverse()                 # தலைகீழாக்கல் (ஒழுங்கல்ல!)
print(m)                    # [10, 20, 30, 40]

n = [3, 1, 2]
print(sorted(n))            # [1, 2, 3]  — புதிய பட்டியல்
print(n)                    # [3, 1, 2]  — மூலப் பட்டியல் மாறவில்லை!

5) இணைத்தல், மீள்பெருக்கம், துண்டாக்கல், படியெடுத்தல்:

a = [1, 2]
b = [3, 4]

print(a + b)           # [1, 2, 3, 4]  (இணைத்தல்)
print(a * 3)           # [1, 2, 1, 2, 1, 2]  (மீள்பெருக்கம்)

m = [10, 20, 30, 40, 50]
print(m[1:4])          # [20, 30, 40]  (துண்டாக்கல்)
print(m[::-1])         # [50, 40, 30, 20, 10]  (தலைகீழ் நகல்)

c = m.copy()           # தனிப் படி (copy)
c.append(60)
print(m)               # [10, 20, 30, 40, 50] — மூலம் பாதிக்கப்படவில்லை
print(c)               # [10, 20, 30, 40, 50, 60]
பரீட்சைப் பொறி d = m என எழுதினால் படி உருவாகாது — இரு பெயர்களும் ஒரே பட்டியலையே சுட்டும்; d ஐ மாற்றினால் m உம் மாறும்! தனிப் படிக்கு m.copy() அல்லது m[:] பயன்படுத்துக. மேலும்: x = m.sort() → x இல் None சேமிக்கப்படும் (sort() எதையும் திருப்பித் தராது).

பட்டியலினூடாக மடக்கு — மாதிரி நிரல்

# வகுப்பின் சராசரிப் புள்ளியும் சித்தியடைந்தோர் எண்ணிக்கையும்
marks = [78, 85, 62, 91, 35, 55]

total = 0
passed = 0
for m in marks:
    total = total + m
    if m >= 40:
        passed = passed + 1

print("சராசரி =", total / len(marks))
print("சித்தியடைந்தோர் =", passed)
# பயனரிடம் இருந்து N புள்ளிகளைப் பெற்று பட்டியலில் சேமித்தல்
marks = []
N = int(input("மாணவர் எண்ணிக்கை: "))
for i in range(N):
    m = int(input("புள்ளி: "))
    marks.append(m)
print("அதிக புள்ளி =", max(marks))
append() — மிக முக்கியம் வெற்றுப் பட்டியலில் தொடங்கி, மடக்கினுள் append() மூலம் உறுப்புகளைச் சேர்ப்பது A/L நடைமுறைப் பரீட்சைகளில் அடிக்கடி வரும் அமைப்பு.
05டியூப்பிள் (Tuple)

பிறை அடைப்புக்குறிகளுக்குள் ( ) எழுதப்படும் வரிசை. பட்டியல் போன்றதே — ஆனால் மாற்ற முடியாதது (immutable). மாறக்கூடாத தரவுகளுக்கு (எ.கா: ஆள்கூறுகள், மாதப் பெயர்கள், RGB நிறங்கள்) ஏற்றது.

point = (9.66, 80.01)          # யாழ்ப்பாணத்தின் ஆள்கூறுகள்
months = ("Jan", "Feb", "Mar")
single = (5,)                  # ஒற்றை உறுப்பு — காற்புள்ளி கட்டாயம்!

print(point[0])      # 9.66
print(len(months))   # 3
print(months[1:3])   # ('Feb', 'Mar')

point[0] = 10        # பிழை! TypeError — tuple ஐ மாற்ற முடியாது

பொதி அவிழ்த்தல் (Unpacking)

student = (1001, "Amuthan", 78)
no, name, marks = student        # ஒவ்வொரு மாறிக்கும் பிரித்து வழங்கல்
print(name)                      # Amuthan

டியூப்பிள் முறைகள் & செயற்பாடுகள் — விரிவான எடுத்துக்காட்டுகள்

1) இரண்டே முறைகள் — count() & index():

t = (5, 8, 5, 3, 5, 9)

print(t.count(5))      # 3   (5 எத்தனை முறை?)
print(t.index(3))      # 3   (முதல் 3 இன் இட எண்)
print(t.index(5))      # 0   (முதல் 5 இன் இட எண்)

2) உள்ளமைச் சார்புகளும் செயலிகளும்:

t = (40, 75, 20, 90)

print(len(t))          # 4
print(max(t))          # 90
print(min(t))          # 20
print(sum(t))          # 225
print(75 in t)         # True
print(t[1:3])          # (75, 20)  (துண்டாக்கல்)
print(t[-1])           # 90

3) இணைத்தலும் மீள்பெருக்கமும் (புதிய tuple உருவாகும்):

a = (1, 2)
b = (3, 4)

c = a + b              # இணைத்தல்
print(c)               # (1, 2, 3, 4)

print(a * 3)           # (1, 2, 1, 2, 1, 2)

# t ஐ "மாற்ற" வேண்டுமெனில் — புதிய tuple உருவாக்குவதே வழி:
t = (10, 20, 30)
t = t + (40,)          # புதிய tuple; பழையது மாற்றப்படவில்லை
print(t)               # (10, 20, 30, 40)

4) வகை மாற்றம் — tuple ↔ list:

t = (40, 10, 30)

m = list(t)            # tuple → list (இப்போது மாற்றலாம்)
m.sort()
t = tuple(m)           # list → tuple (மீண்டும் பூட்டல்)
print(t)               # (10, 30, 40)

print(sorted(t))       # [10, 30, 40] — கவனம்: sorted() எப்போதும்
                       # list ஐயே தரும், tuple ஐ அல்ல!
  • டியூப்பிளில் count(), index() ஆகிய இரு முறைகளே உள்ளன — append / remove / sort போன்றவை இல்லை.
  • len(), max(), min(), sum(), slicing, in — அனைத்தும் பயன்படுத்தலாம்.
  • தரவுத்தள இணைப்பில் fetchall() தரும் ஒவ்வொரு பதிவும் ஒரு tuple — இரு பாடப்பகுதிகளின் தொடர்பு!
பரீட்சைப் பொறி (5) என்பது tuple அல்ல — வெறும் எண் 5. ஒற்றை உறுப்பு tuple க்கு (5,) எனக் காற்புள்ளி கட்டாயம்.
06அகராதி (Dictionary)

நெளி அடைப்புக்குறிகளுக்குள் { } எழுதப்படும் சாவி : பெறுமானம் (key : value) சோடிகளின் தொகுப்பு. உறுப்புகள் இட எண்ணால் அல்ல, சாவியால் அணுகப்படும். சாவிகள் தனித்துவமானவை (unique); பெறுமானங்களை மாற்றலாம் (mutable).

student = {"no": 1001, "name": "Amuthan", "marks": 78}

print(student["name"])     # Amuthan  (சாவி மூலம் அணுகல்)

student["marks"] = 85      # பெறுமானம் திருத்தல்
student["grade"] = "A"     # புதிய சோடி சேர்த்தல்
del student["no"]          # சோடி நீக்கல்

print(student)
# {'name': 'Amuthan', 'marks': 85, 'grade': 'A'}

முக்கிய முறைகள்

முறைதருவது
d.keys()எல்லாச் சாவிகளும்
d.values()எல்லாப் பெறுமானங்களும்
d.items()(சாவி, பெறுமானம்) சோடிகள்
d.get("name")பெறுமானம் (சாவி இல்லாவிடின் None — பிழை எழுப்பாது)
len(d)சோடிகளின் எண்ணிக்கை
"name" in dசாவி உள்ளதா (True / False)

அகராதி முறைகள் — விரிவான எடுத்துக்காட்டுகள்

1) அணுகல் முறைகள் — [ ] மற்றும் get():

d = {"no": 1001, "name": "Amuthan", "marks": 78}

print(d["name"])           # Amuthan
print(d.get("marks"))      # 78
print(d.get("grade"))      # None  (சாவி இல்லை — ஆனால் பிழை இல்லை)
print(d.get("grade", "X")) # X     (இல்லாவிடின் கொடாநிலைப் பெறுமானம்)
# print(d["grade"])        # KeyError! — [ ] முறையில் பிழை எழும்

2) keys(), values(), items():

d = {"ICT": 85, "Physics": 72, "Maths": 90}

print(list(d.keys()))      # ['ICT', 'Physics', 'Maths']
print(list(d.values()))    # [85, 72, 90]
print(list(d.items()))     # [('ICT', 85), ('Physics', 72), ('Maths', 90)]

print(len(d))              # 3
print("ICT" in d)          # True   (சாவியைச் சோதிக்கும்)
print(85 in d)             # False  (பெறுமானத்தை அல்ல!)
print(85 in d.values())    # True   (பெறுமானச் சோதனை இவ்வாறு)
print(max(d.values()))     # 90
print(sum(d.values()))     # 247

3) சேர்த்தல் / திருத்தல் — update():

d = {"a": 1, "b": 2}

d["c"] = 3                     # ஒரு சோடி சேர்த்தல்
d.update({"d": 4, "b": 20})    # பல சோடிகள் — b ஏற்கனவே இருப்பதால் திருத்தப்படும்
print(d)                       # {'a': 1, 'b': 20, 'c': 3, 'd': 4}

4) நீக்கும் முறைகள்:

d = {"a": 1, "b": 2, "c": 3, "d": 4}

x = d.pop("b")         # சாவி b ஐ நீக்கி, பெறுமானத்தைத் தரும்
print(x, d)            # 2 {'a': 1, 'c': 3, 'd': 4}

y = d.popitem()        # கடைசிச் சோடியை நீக்கித் தரும்
print(y, d)            # ('d', 4) {'a': 1, 'c': 3}

del d["a"]             # del மூலம் நீக்கல்
print(d)               # {'c': 3}

d.clear()              # எல்லாச் சோடிகளையும் நீக்கல்
print(d)               # {}

5) படியெடுத்தல் — copy():

d = {"a": 1, "b": 2}
e = d.copy()           # தனிப் படி
e["c"] = 3
print(d)               # {'a': 1, 'b': 2} — மூலம் மாறவில்லை
print(e)               # {'a': 1, 'b': 2, 'c': 3}
பரீட்சைப் பொறி "x" in d — சாவிகளில் மட்டுமே தேடும்; பெறுமானங்களில் தேட x in d.values() எழுத வேண்டும். இல்லாத சாவியை d["x"] என அணுகினால் KeyError; d.get("x") பாதுகாப்பானது.

அகராதியினூடாக மடக்கு — மாதிரி நிரல்

# பாடங்களும் புள்ளிகளும்
results = {"ICT": 85, "Physics": 72, "Maths": 90}

for subject in results:                 # சாவிகளினூடாக
    print(subject, "→", results[subject])

for sub, mark in results.items():       # சோடிகளினூடாக
    print(sub, "=", mark)

print("மொத்தம் =", sum(results.values()))
எப்போது அகராதி? தரவுகளை பெயரால் / அடையாளத்தால் தேட வேண்டுமெனில் அகராதி; வரிசை முக்கியம் எனில் பட்டியல். எ.கா: சுட்டெண் → பெயர் தேடல் = அகராதி; வந்த வரிசையில் புள்ளிகள் = பட்டியல்.

08பரீட்சைக் குறிப்புகள் — பொதுவான பொறிகள்
  • இட எண் 0 இல் தொடங்கும்; கடைசி உறுப்பின் இட எண் len - 1 (அல்லது -1).
  • துண்டாக்கலில் [a:b] — b சேராது; உறுப்புகளின் எண்ணிக்கை = b − a.

  • “வெளியீடு என்ன?” (predict the output) வினாக்களில் ஒவ்வொரு வரியின் பின்னும் கட்டமைப்பின் நிலையை எழுதி (trace) செல்வது பிழைகளைத் தவிர்க்கும்.

வினா 05

ஒரு கடையின் பொருட்களும் விலைகளும் அகராதியில் உள்ளன: items = {"அரிசி": 240, "சீனி": 280, "பால்மா": 1150}. பின்வருவனவற்றைச் செய்யும் நிரல் வரிகளை எழுதுக:
(அ) "சீனி" இன் விலையை 260 ஆகத் திருத்துக  (ஆ) "தேயிலை": 950 எனப் புதிய பொருளைச் சேர்க்குக  (இ) "பால்மா" ஐ நீக்குக  (ஈ) எல்லாப் பொருட்களையும் விலைகளுடன் அச்சிடுக

விடை 
items = {"அரிசி": 240, "சீனி": 280, "பால்மா": 1150}

items["சீனி"] = 260          # (அ) திருத்தல்
items["தேயிலை"] = 950        # (ஆ) சேர்த்தல்
del items["பால்மா"]          # (இ) நீக்கல்
