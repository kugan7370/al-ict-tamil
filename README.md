# AL ICT — Tamil Revision Hub

A static revision website for the G.C.E. Advanced Level **Information & Communication
Technology (ICT)** curriculum, written in **Tamil**. It covers the theory units, Python
programming, data structures, algorithms, and database topics, with interactive
click-to-reveal answers.

Pure HTML/CSS/JavaScript — no build step. Open `index.html` locally, or visit the live
site:

**🔗 Live:** https://kugan7370.github.io/al-ict-tamil/
